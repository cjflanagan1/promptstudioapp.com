Frequently Asked Questions

Which AI providers are supported?

AI Prompt Studio works with a wide range of providers. Bring your own API key for any of these services:

  • Anthropic (Claude)
  • OpenAI (GPT)
  • xAI (Grok)
  • Google (Gemini)
  • Mistral
  • DeepSeek
  • Perplexity
  • Apple Intelligence (on-device, no key required)

Where is my data stored?

All your data stays on your device. Prompts, templates, vault documents, and preferences are stored locally. API keys are secured in the iOS Keychain. Nothing is sent to our servers — we don't have any.

What subscription tiers are available?

AI Prompt Studio offers a free tier with core features and a Pro subscription that unlocks the full experience, including unlimited prompts, all AI providers, and advanced vault capabilities. All payments are handled securely through Apple.

What file types can I import into the Vault?

The Vault supports a variety of document formats for context attachment, including PDF, plain text, Word documents, and more. Text extraction happens entirely on-device for your privacy.

Do I need an internet connection?

You need an internet connection to send prompts to cloud-based AI providers. However, Apple Intelligence runs on-device and works offline. Building and organizing prompts can be done without a connection.

How do I add my API key?

Go to Settings within the app, select your AI provider, and paste your API key. It's stored securely in the iOS Keychain and is only sent to the provider's API when you run a prompt.
